## Get Withdrawal

Retrieve a withdrawal by its ID. Poll this endpoint to track progress from `pending_deposit` through `completed`.

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key.
</ParamField>

### Path parameters

<ParamField path="id" type="string" required>
  The withdrawal UUID.
</ParamField>

<RequestExample>
```bash
curl https://kola-api-beta.vercel.app/api/v1/withdrawals/10f98e4d-83c5-4efb-adff-3cd82b2e9dac \
  -H "X-API-Key: YOUR_API_KEY"
```
</RequestExample>

### Response fields

<ResponseField name="status" type="string">
  One of: `pending_deposit`, `received`, `processing`, `completed`, `failed`, `expired`.
</ResponseField>

<ResponseField name="solanaTxSignature" type="string | null">
  The signature of the user's USDC transfer to the treasury. Populated when status reaches `received` or later.
</ResponseField>

<ResponseField name="monimePayoutId" type="string | null">
  Monime's payout identifier. Populated when the payout is dispatched (status `processing` or later).
</ResponseField>

<ResponseField name="failureReason" type="string | null">
  Error detail when status is `failed`.
</ResponseField>

### Errors

| Status | Message |
|--------|---------|
| 401 | `Missing API key` / `Invalid API key` |
| 404 | `Withdrawal not found` |

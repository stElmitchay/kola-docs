## List Deposits

Returns a list of deposits for your application. Supports filtering by phone number and status.

### Headers

<ParamField header="X-API-Key" type="string" required>
  Your API key.
</ParamField>

### Query parameters

<ParamField query="phoneNumber" type="string">
  Filter by phone number (normalized format, e.g., `+23276123456`).
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `pending`, `confirmed`, `processing`, `completed`, `failed`, `expired`.
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of results (default: all).
</ParamField>

<ParamField query="offset" type="integer">
  Number of results to skip (for pagination).
</ParamField>

<RequestExample>
```bash All deposits
curl https://kola-api-beta.vercel.app/api/v1/deposits \
  -H "X-API-Key: YOUR_API_KEY"
```

```bash Filtered
curl "https://kola-api-beta.vercel.app/api/v1/deposits?status=completed&limit=10" \
  -H "X-API-Key: YOUR_API_KEY"
```
</RequestExample>

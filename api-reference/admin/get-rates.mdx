## Get Exchange Rates

Returns the current active rate and the full rate history for an application.

### Headers

<ParamField header="X-Admin-Key" type="string" required>
  Your admin secret key.
</ParamField>

<ParamField header="X-Application-Id" type="string" required>
  The application ID.
</ParamField>

<RequestExample>
```bash
curl https://kola-api-beta.vercel.app/api/v1/admin/exchange-rates \
  -H "X-Admin-Key: YOUR_ADMIN_KEY" \
  -H "X-Application-Id: a7784b0d-8be0-4338-be20-1d1a3ac9d1fc"
```
</RequestExample>

<ResponseExample>
```json
{
  "current": {
    "id": "c3d4e5f6-7890-1abc-def2-345678901bcd",
    "application": "a7784b0d-8be0-4338-be20-1d1a3ac9d1fc",
    "rate": 25,
    "is_active": true,
```

### Response fields

<ResponseField name="current" type="object | null">
  The currently active exchange rate. `null` if no rate has been set.
</ResponseField>

<ResponseField name="history" type="array">
  All exchange rates for this application, ordered by creation date (newest first).
</ResponseField>

### Errors

| Status | Message |
|--------|---------|
| 400 | `X-Application-Id header is required` |
| 401 | `Missing admin key` / `Invalid admin key` |

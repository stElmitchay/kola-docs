## Create API Key

Generates a new API key. The raw key is returned **once** — store it securely.

### Headers

<ParamField header="X-Admin-Key" type="string" required>
  Your admin secret key.
</ParamField>

### Body parameters

<ParamField body="name" type="string" required>
  A label for this API key (e.g., `"Production Key"`).
</ParamField>

<ParamField body="businessName" type="string" required>
  The business or organization name.
</ParamField>

<ParamField body="appName" type="string" required>
  The application name.
</ParamField>

<ParamField body="expiresAt" type="string">
  ISO date when this key should expire. Omit for no expiration.
</ParamField>

<ParamField body="permissions" type="object">
  Custom permission flags. Defaults to `{}`.
</ParamField>

<ParamField body="rateLimit" type="integer">
  Requests per minute. Defaults to `100`.
</ParamField>

<RequestExample>
```bash
curl -X POST https://kola-api-beta.vercel.app/api/v1/admin/api-keys \
  -H "Content-Type: application/json" \
  -H "X-Admin-Key: YOUR_ADMIN_KEY" \
  -d '{
    "name": "Production Key",
    "businessName": "Acme Corp",
    "appName": "Acme Wallet"
  }'
```
</RequestExample>

<ResponseExample>
```json
{
  "message": "API key created. Store this key securely — it will not be shown again.",
  "apiKey": "kola_9afab24a3610acdb5bb9f9a990d45daee4dc2395dde3faae1c5ee559401c4dbd",
  "application": {
    "id": "a7784b0d-8be0-4338-be20-1d1a3ac9d1fc",
    "client_id": "b2e3f4a5-6789-0abc-def1-234567890abc",
    "name": "Production Key",
    "business_name": "Acme Corp",
    "app_name": "Acme Wallet",
    "is_active": true,
    "rate_limit": 100,
    "permissions": {},
    "created_at": "2026-05-16T18:30:00.000Z"
  }
}
```
</ResponseExample>

### Response fields

<ResponseField name="apiKey" type="string">
  The raw API key. **Store this immediately** — it cannot be retrieved again. The key is prefixed with `kola_`.
</ResponseField>

<ResponseField name="application" type="object">
  The created application record, including the `id` you'll need for setting exchange rates.
</ResponseField>

### Errors

| Status | Message |
|--------|---------|
| 401 | `Missing admin key` / `Invalid admin key` |
